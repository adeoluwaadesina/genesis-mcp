# Genesis MCP — Demo Walkthroughs

These examples show the three main tool archetypes you can create with `create_tool`.
Copy the description text into Claude Desktop and call `create_tool` with it.

| Demo | Archetype | File |
|---|---|---|
| `get_weather` | External API (no key) | [weather_demo.md](weather_demo.md) |
| `save_note` | Filesystem write | [notes_demo.md](notes_demo.md) |
| `convert_currency` | Data transformation + external API | [currency_demo.md](currency_demo.md) |
